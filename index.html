Game Modding

I got into game modding when I was in middle school. It started as just altering ini files, but over the years I've wanted to learn more and more. It's helped me learn a lot about memory structures in games and how to alter them to insert new content compatible with the engine. Just one example of these is rom hacking. It's an old form of modding where retro video game roms have their memory altered to allow whole new levels. While by no means comprehensive, here's a website with a large database of rom hacks for a large collection of retro games: romhacking.net
